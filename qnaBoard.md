## QnA 게시판 화면 시연 (관리자 / 일반 회원 권한 차등)
- 전체 QnA 게시물 조회, 작성은 모두 가능하게 설정
- QnA 상세 페이지에서 일반 회원은 댓글 조회만 가능
- 관리자 계정으로 접속 시에만 댓글 작성 버튼 노출
- 본인이 작성한 게시물만 수정, 삭제 가능
